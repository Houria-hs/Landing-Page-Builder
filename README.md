üåü Project Overview

[MVP] The Landing Page Builder is currently a Minimum Viable Product focused on delivering a foundational, component-based visual editor. It allows users to quickly assemble and preview simple, responsive landing pages using a core set of pre-built blocks. The primary goal of this MVP is to validate the drag-and-drop editing experience and secure user data through robust authentication, paving the way for advanced features like Auto Smart Alignment and Real-Time Collaboration in future releases.

This project is built with a modern React architecture, utilizing a component-based system and Tailwind CSS for rapid, utility-first styling.

‚ú® Key Features

User Authentication & Profiles: Secure sign-up and login capabilities with persistent user profiles for managing saved personal user data .

Drag-and-Drop Interface: Intuitive visual editor for arranging and structuring page components.

Real-time Preview: See exactly what your page looks like as you build it.

Component Library: Includes essential building blocks like Navbar, Header, About Us, Portfolio, Form, and Footers.

Fully Responsive: All generated pages are automatically optimized for desktop, tablet, and mobile viewing.

Customizable Styling: Comprehensive control over component appearance, including colors, custom widths and heights, adjustable font sizes, font family selection via Google Fonts API integration, bold toggles, background colors, and color fills.

Export Functionality : Ability to export the final page as clean HTML and CSS.

üöÄ Usage

Start Building: Once the application is running, navigate to the main editor interface.

Add Components: Use the sidebar to drag a new component (e.g., a "Hero Section" or "Portfolio Section") onto the canvas.

Edit Content: Click directly on any text or image area on the canvas to modify the content.

Configure Styles: Use the properties panel to change backgrounds, fonts, and size for the selected component.

Export:export the finished product as HTML. 


üíª Built With

Frontend:

React: Core frontend library for building the UI.

Tailwind CSS: Utility-first framework for rapid, responsive styling.

Framer Motion: Used for smooth animations and transitions within the editor interface.

Lucide React: Icon library for clean UI elements.

Backend & Data:

Backend Repository: https://github.com/Houria-hs/Landing-Page-Builder-backend

Node.js (Express): Server-side environment for handling API routes and application logic.

JWT (JSON Web Tokens): Used for secure, stateless user authentication and session management.

PostgreSQL: Relational database for persistent storage of user data, projects, and templates.
